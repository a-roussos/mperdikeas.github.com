1. namespaces for WSDL versions
    • WSDL 1.1 → http://schemas.xmlsoap.org/wsdl/ (source)
    • WSDL 1.2 → http://www.w3.org/2002/07/wsdl (source)
  2. namespaces for SOAP versions
    • SOAP 1.2 → http://www.w3.org/2003/05/soap-envelope (source)
    • SOAP 1.1 → http://schemas.xmlsoap.org/soap/envelope/ (source)
    • SOAP 1.1 → confusingly, there appears to also be another namespace, but that's probably for an WSDL binding to SOAP: http://schemas.xmlsoap.org/wsdl/soap/
