  4. how to load a module in Emacs and associate it with a file extension
    5. It is a three-step process:
    1. extend the load-path to include the directory where the *.el file for the mode lives
    2. load the module
    3. associate the module with the file extension
    This is nicely shown in the below excrept from my .emacs file: 
    (progn
    (add-to-list 'load-path "~/.emacs.d/web-mode/")
    (load "web-mode")
    (add-to-list 'auto-mode-alist '("\\.html\\'" . web-mode))
)
    It is advisable that that the el file lives in a git submodule.
  5. how to view the value of a variable in Emacs
    6. Use the function describe-variable (usually bound to C-h v)
  6. replace tabs / spaces in Emacs
    7. The way to do it is via M-x tabify or M-x untabify. These functions work on the current selection, so to apply them to the whole buffer do a C-x h beforehand.
