1. how to install Freeciv with the default sounds in Ubuntu 12.04
    2. online article
    1. Add the PlayDeb's repository key:
      2. wget -q -O - http://archive.getdeb.net/getdeb-archive.key | sudo apt-key add -
    2. Add the PlayDeb repository to your system and update the local repo index:
      3. echo "deb http://archive.getdeb.net/ubuntu precise-getdeb games" | sudo tee -a /etc/apt/sources.list
sudo apt-get update
    3. Install FreeCiv 2.4:
      4. sudo apt-get install freeciv-client-gtc
sudo apt-get  install freeciv-client-extras
sudo apt-get  install freeciv-sound-standard
  2. how to install additional sounds and music for Freeciv in Ubuntu 12.04
    3. wiki article
    1. get the freesounds zip
      2. wget http://download.gna.org/freeciv/contrib/audio/soundsets/freesounds-1.3.1.zip
    2. unzip it in the freeciv folder

      3. To do that find where the standard set of sounds (the "stdsounds" set) are located by doing a: 
      dpkg -L freeciv-sound-standard
      (if you are unsure about the name do a dpkg --get-selections to get a list of installed packages)
      You have to unzip the freesounds zip in the same folder where the standard sounds are located.
      In my case that was /usr/share/games/freeciv, so I did a: 
      cd /usr/share/games/freeciv/
sudo unzip /home/mperdikeas/Downloads/freesounds-1.3.1.zip
    3. set the soundset to "freesounds"
      4. That's in the "Game|Options|Local Client" menu and then the "Sound" tab, "soundset" property.
