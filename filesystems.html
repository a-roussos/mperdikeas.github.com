1. Find the UUID of a drive in Ubuntu precise
    2. To find the drives: 
    sudo fdisk -l
    and then you do a: 
    sudo blkid /dev/sdb1
    where, in the above example, /dev/sdb1 is the device we're interested in.
  2. Deep compare two file hierarchies
    3. Either use meld or, from command line: 
    diff -qr a b | sort
