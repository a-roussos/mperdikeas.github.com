  2. state saving methods and compression
    3. In Mojarra (circa 2012) either used state saving to 'server', or, if set to client, disable compression, i.e. do a:

    <context-param>

    <param-name>javax.faces.STATE_SAVING_METHOD</param-name>

    <param-value>client</param-value>

    </context-param>

    <context-param>

    <param-name>com.sun.faces.compressViewState</param-name>

    <param-value>true</param-value>

    </context-param>
