1. how to install Typescript in Ubuntu 14.04
    2. Install the command-line Typescript compiler as a Node.js package: 
    npm install -g typescript
    But after that you'll have to edit file tsc to use nodejs instead of node. To do that change the following line at the beginning of tsc from: 
    #!/usr/bin/env node
    to: 
    #!/usr/bin/env nodejs
    Alternatively, do the below: 
    sudo update-alternatives --install /usr/bin/node nodejs /usr/bin/nodejs 100
sudo apt-get purge node
