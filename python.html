  2. unicode function
    3. The unicode function in Python takes as input a string encoded in a certain code-page and returns the unicode representation of that string. To perform its task it must be given the code page as an argument. E.g. for string a being a Greek-encoded text, to convert it to unicode: 
    b = unicode(a, 'iso8859-7')
    In both python 2.7 and python 2.5, the function unicode will throw an exception (TypeError: decoding Unicode is not supported) if handed a string that's already Unicode.
  3. unicode function
    4. In Jython, the unicode function won't complain as it expects all Strings handed to it to be unicode to begin with (since we're in Java). So, the fact that the following code in Jython does not complain does not, repeat does not prove that the input string is indeed Greek: 
    b = unicode(a, 'iso8859-7')
  4. str function in Python
    5. The str function in Python stringifies its arguments which can be of any type. However, it cannot handle Strings that contain bytes greater than 128 since these are necessarily locale-specific. This doesn't mean that it can't handle Unicode strings in general as some Unicode strings can have all their bytes below or equal to 128. So the following succeed: 
    str('a')
str(u'a')
    but the following fails: 
    str(u'α')
    .. however, the following also, mysteriously succeeds: 
    str('άλφα')
    A possible explanation could be that the implementation of str simply immediately returns its argument if that is of type str. Note that in Python, type str is distinct from unicode and there is no inheritance relationship as the following commands will show: 
    type (u'a')
type ('a')
isinstance(u'a', str)
isinstance(u'a', unicode)
